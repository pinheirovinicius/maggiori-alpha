# Maggiori Alpha - Site (GitHub Pages)

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub (ex.: `maggiori-alpha`)
2. Envie os arquivos deste projeto (mantendo a estrutura)
3. No GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **(root)**
4. Aguarde o deploy e acesse a URL do Pages.

## Ajustar o sitemap
Edite `sitemap.xml` e substitua:
- `https://SEU-USUARIO.github.io/SEU-REPO/`
pela URL real do seu GitHub Pages.
